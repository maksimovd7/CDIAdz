

### Массивы (списки)

1) Размер  
В языке C++ размер массива задается сразу при объявлении, изменять его после компиляции нельзя — придется создавать новый массив и переносить в него данные. В Java длина массива фиксируется при создании: чтобы расширить или сократить массив, нужно создать другой и скопировать элементы. В Python списки динамически изменяют размер — можно свободно добавлять и удалять элементы без явного выделения памяти.

2) Типы данных  
В C++ и Java массивы могут содержать только данные одного определённого типа, указанного при объявлении. В Python элементы списка могут быть разных типов одновременно — можно комбинировать числа, строки, булево значения и так далее.

3) Встроенные операции  
Python предлагает широкий набор встроенных методов для работы со списками: добавление, удаление, сортировка, поиск и множество других операций. В C++ для таких задач зачастую требуется самостоятельно реализовать алгоритмы или обратиться к сторонним библиотекам. В Java для сортировки, поиска и копирования элементов применяются специальные вспомогательные классы и методы.

### Стек

1) Реализация  
В C++ стек обычно создают на базе массива с фиксированным размером — это экономит память, но ограничивает количество элементов. В Java есть готовый класс Stack, реализующий принцип Last In — First Out (LIFO); можно использовать его или создавать свой аналог. В Python стек реализуют с помощью списка или отдельного класса, который хранит логику работы стека.

2) Операции  
Во всех трех языках для добавления элемента наверх используется операция push, для удаления — pop. Освобождать память вручную не требуется ни в одном из языков: она очищается автоматически.
